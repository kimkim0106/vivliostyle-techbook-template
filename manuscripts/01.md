# 第1章 サンプル章

## 見出し

ここに本文を書きます。このテンプレートでは、段落の先頭行には字下げが入るようになっています。

段落を変えるには、空行を入れる必要があります。

## VFM (Vivliostyle Flavored Markdown)

基本的に Markdown の記法がそのまま使えます。

Vivliostyle 独自の構文もあるため、詳しくは、[VFM の公式ドキュメント](https://vivliostyle.github.io/vfm/#/vfm)を参照してください。

```ts:index.ts
// コードブロックも使えます (例: JavaScript)
function main() {
    console.log("Hello, Vivliostyle!");
    return;
}
```

図番号を参照できるようになっています。たとえば <a href="#logo" data-ref="fig"></a> のような感じです。

### .textlintrc

日本語の文章を書くうえで便利なルールを設定しています。詳しい説明は各ルールのドキュメントを参照してください。

- preset-ja-spacing — 半角文字と全角文字の間にスペースを入れるルール
- preset-japanese — 日本語の一般的な文章ルール（二重否定、ら抜き言葉、同じ接続詞の連続使用など）
- preset-ai-writing — AI ライティング向けのプリセット
- prh — 校正ヘルパー（ICS MEDIA の辞書を利用）

また、フィルターも設定しています。

- comments — コメントで囲むことで、特定の範囲のルールを一時的に無効化できます
- allowlist — `.textlintrc` の `allow` 配列に文字列を追加すると、その文字列をチェック対象外にできます

```markdown
<!-- textlint-disable ja-spacing/ja-space-between-half-and-full-width -->
ここはチェックされない
<!-- textlint-enable ja-spacing/ja-space-between-half-and-full-width -->
```

チェックの実行と自動修正は以下のコマンドで行えます。

```bash
npm run textlint        # チェック
npm run textlint:fix    # 自動修正
```
